# Elf File Analyzer
A windows utility tool for analyzing elf, hex and binary firmware files

## Dependency Libraries (.Net Standard)
- https://github.com/gsmrana/Megamind.IO.FileFormat
- https://github.com/gsmrana/Megamind.Num.Encode
